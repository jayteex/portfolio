React Reddit JSON API App

This project, codename "rddt minimal", fetches data from the reddit JSON API. The user can specify any subreddit of their choosing, and the corresponding data will be fetched. The user can then search the fetched posts in real time, then choose a story that is most interesting to them and get redirected direclty to it.

(Due to the wildly inconsistent nature of the JSON API, the only reliable image source that all posts share is the thumbnail. Image quality is therefore limited.)

Technologies used

  • React
  • Redux (Toolkit)
  • TypeScript
  • JavaScript
  • CSS
    • Grid
    • Flexbox
    • Modules
  • HTML
  • Vite
